# myfirstprojectArsenalform
## Arsenal FC Inspired Form
This is a simple form inspired by the design of Arsenal FC. It has been created using HTML and CSS.

### Usage
To use this form, simply include the HTML code in your webpage and link to the CSS file. You can modify the form fields and styling as necessary to fit your specific needs.

### Features
The form includes fields for name, email, and message.
The form has been styled using Arsenal FC's signature red and white colors.
The form includes a button styled in Arsenal FC's font and color scheme.
